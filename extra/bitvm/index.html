BitVM

Links
  • homepage

Introduction

BitVM is an optimistic generalized computation protocol that can be deployed on bitcoin today. It is equivalent to the MATT protocol, but with the need for covenants replaced with both a reduced security assumption and a much increased complexity.

Implementations

An official implementation is being developed in Rust.
